QUICK START:
════════════

Windows:  Double-click setup.bat
Linux/Mac: bash setup.sh

Then run in separate terminals:
  Terminal 1: cd backend && python manage.py runserver
  Terminal 2: cd frontend && npm start

Open: http://localhost:3000
